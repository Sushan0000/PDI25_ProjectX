# Strike Protocol; Unity TPS Game

A TPP inspired by **Counter-Strike** and **Call of Duty**, **PUBG** built in Unity. 
Fast-paced tactical combat with realistic weapons and responsive controls.

## Overview
- **Engine:** Unity
- **Language:** C#  
- **Genre:** Third-Person Shooter
- **Mode:** Deathmatch  
- **Platform:** PC / Android / iOS
