# Strike Protocol; Unity FPS Game

A first-person shooter inspired by **Counter-Strike** and **Call of Duty**, built in Unity.  
Fast-paced tactical combat with realistic weapons, responsive controls, and multiplayer-ready architecture.

## Overview
- **Engine:** Unity
- **Language:** C#  
- **Genre:** First-Person Shooter (FPS)  
- **Mode:** Team Deathmatch / Free-for-All  
- **Platform:** PC (Windows) / Android / iOS
